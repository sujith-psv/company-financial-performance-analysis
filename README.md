# Company Financial Performance and Valuation Analysis

## Overview
This project analyzes company financial performance and market valuation using Python and Power BI.
Operational efficiency is evaluated using operating margins, while valuation is assessed using P/E and P/B ratios.

## Workflow
- Cleaned and transformed financial data using Python (Pandas)
- Calculated profitability and valuation metrics
- Classified companies by performance and valuation categories
- Built an interactive Power BI dashboard for visualization

## Tools Used
- Python (Pandas, NumPy)
- Power BI
- Excel

## Key Insights
- Strong operational performance generally aligns with higher market valuation
- Some moderately performing companies trade at valuation premiums
- Financially weak companies are consistently penalized by the market
